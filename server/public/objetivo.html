Objetivo del proyecto

Desarrollar un sitio web utilizando bootstrap y html para la maquetación y diseño del mismo, node.js, javascript y php para la programación, donde su funcionamiento será mostrar en tiempo real los datos que recibirá el sensor ultrasónico que mide la distancia de los obstáculos que se encuentran cerca de él.
